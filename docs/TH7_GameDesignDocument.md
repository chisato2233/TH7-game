## 1. 游戏概述

### 1.1 核心玩法
玩家将在泰拉诺斯大陆上选择一个文明，建立势力、招募英雄、训练军队、研究魔法，最终收集七块"神之守护碎片"，决定大陆的命运。

### 1.2 游戏特色
- **七大文明并立**: 阿拉伯、埃及、印度、希腊、汉唐、蒙古、南岛，每个文明拥有独特的兵种、英雄、魔法和神话生物
- **神话源流魔物**: 摒弃西方奇幻套路，从中国、埃及、印度、南岛神话中汲取邪恶力量原型
- **跨文明融合魔法**: 不同文明的魔法可以组合，产生更强大的融合魔法
- **深度事件系统**: 每个文明50个等级事件，根据玩家的学识、魔法、军力等级解锁

### 1.3 核心循环
```
探索世界 → 招募英雄 → 建设势力 → 研究魔法 → 征服/外交 → 收集碎片 → 决定命运
```

## 2. 世界观设定

### 2.1 创世神话
泰拉诺斯大陆由**七大神祇**共同守护，而第八位神祇**普罗米修斯**因窃取创世之火被其他七神吞噬。普罗米修斯的力量化为"父神碎片"散落世间，这些碎片会扭曲生物，创造出魔物。

### 2.2 七大守护神

| 神祇 | 领域 | 对应文明 | 赋予魔法 |
|------|------|----------|----------|
| 阿努比斯 | 死亡、守护 | 埃及 | 死灵魔法、守护结界 |
| 伊南娜 | 战争、爱情 | 阿拉伯 | 战斗强化、魅惑术 |
| 湿婆 | 毁灭、重生 | 印度 | 破坏魔法、治愈再生 |
| 宙斯 | 雷电、统治 | 希腊 | 雷电魔法、王者威严 |
| 玉皇大帝 | 天道、秩序 | 汉唐 | 道术、符箓 |
| 长生天 | 自然、萨满 | 蒙古 | 自然魔法、图腾 |
| 塔纳罗阿 | 海洋、创世 | 南岛 | 海洋魔法、天气术 |

### 2.3 核心冲突
- 七神的封印正在崩溃
- 父神碎片正在腐蚀世界
- 各文明为了自保或野心而争夺碎片
- 玩家必须选择：重建秩序，还是拥抱混沌？

## 7. 兵种与神话生物

### 7.1 兵种等级体系

| 等级 | 描述 | 示例 |
|------|------|------|
| 1级 | 基础单位，易招募 | 民兵、斥候 |
| 2级 | 正规军，需兵营 | 步兵、弓兵、轻骑 |
| 3级 | 精锐单位，需训练场 | 重骑、精锐射手 |
| 4级 | 高级单位，需特殊建筑 | 法师、战象 |
| 5级 | 传奇单位，需英雄/任务解锁 | 龙骑士、神话生物 |

### 7.2 克制关系
```
骑兵 > 弓兵 > 步兵 > 骑兵
```

### 7.3 神话生物特性
