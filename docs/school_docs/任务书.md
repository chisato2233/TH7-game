设计（论文）题目
中文：基于Unity引擎的多文明策略游戏设计与实现
英文：Design and Implementation of Multi-Civilization Strategy Game Based on Unity Engine
题目类型 工程实践类 ☑
题目来源 题目是否来源于科研项目：否 ☑
主要内容： 本课题旨在设计并实现一款基于Unity引擎的多文明策略角色扮演游戏。游戏以七大文明（阿拉伯、埃及、印度、希腊、汉唐、蒙古、南岛）并立的奇幻世界为背景，玩家可选择不同文明进行游戏，体验独特的兵种体系、英雄系统、魔法系统和文明事件。 主要研究内容包括：
游戏框架设计：设计模块化的游戏系统架构，包括系统管理器、事件系统、流程状态机等核心模块；
文明系统设计：实现七大文明的差异化设计，包括各文明特色兵种、神话生物、英雄和专属魔法；
战斗系统实现：设计回合制或即时策略战斗机制，实现兵种克制、魔法效果、英雄技能等功能；
数据驱动架构：设计可扩展的配置表系统，支持文明、兵种、英雄、魔法等数据的灵活配置；
用户界面开发：实现游戏主界面、战斗界面、文明管理界面等核心UI模块。
主要（技术）要求：
熟练掌握Unity游戏引擎及C#编程语言；
掌握面向对象设计原则，能够设计模块化、可扩展的游戏架构；
熟悉常用设计模式（单例模式、观察者模式、状态模式等）在游戏开发中的应用；
了解游戏数值设计基础，能够进行基本的数值平衡调整；
具备良好的文档编写能力，能够撰写规范的设计文档和技术文档；
完成核心游戏功能的开发，包括至少2个完整文明的实现；
提交可运行的游戏Demo及完整的源代码和设计文档。
